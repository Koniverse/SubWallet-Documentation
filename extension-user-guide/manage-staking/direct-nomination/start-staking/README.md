# Start staking

**Step 1**: Open SubWallet and choose the Staking tab. Then click the "+" icon on the upper right corner to start staking.

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FRGWvXesNAq3NmUptr0HQ%252Fimage.png%3Falt%3Dmedia%26token%3D64ba993b-3f4f-42b1-a815-1381822a4841\&width=300\&dpr=4\&quality=100\&sign=f51f657a\&sv=1)

**Step 2**: Choose the "Nominate" tab in the "Add to Bond" screen and enter the staking information.

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FpPurRFaj1Ln7OCVJjSky%252Fimage.png%3Falt%3Dmedia%26token%3Dadf53a02-fb4f-45a6-8f95-bb490c05e17a\&width=300\&dpr=4\&quality=100\&sign=bba84f44\&sv=1)

If you are in all-accounts mode like the current example, you would need to choose the staking account.

If you have already been in a single account, you would not have to select account. Your Step 2 would be as the following image:

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252Fo2008qAqhD4lsQYvzVs2%252Fimage.png%3Falt%3Dmedia%26token%3Dd3ec5901-292e-42c5-a40b-65ec3a924876\&width=300\&dpr=4\&quality=100\&sign=1187a45c\&sv=1)

We suggest you pay close attention to the validator you are choosing. When selecting validator, SubWallet support you with the latest record of validator details. Please click the three-dot icon on the right hand side of each validator to see the validator details.

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FbP8SrrRJOzrIprH7uDqL%252Fimage.png%3Falt%3Dmedia%26token%3Da55b0088-8e2d-477a-8267-b9f20318b632\&width=300\&dpr=4\&quality=100\&sign=5af55c16\&sv=1) ![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252F5r6bmFKMoDZgpVnhUfbx%252Fimage.png%3Falt%3Dmedia%26token%3Db7b99c5c-3dfb-4ad0-9603-f22b39bfe30b\&width=300\&dpr=4\&quality=100\&sign=675db0a3\&sv=1)

You could use the sort function to find the most suitable validator according to your need. Please click the sort icon on the upper right corner and choose your sorting criteria.

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FidT1Gw2Rco1eHRsqf3tW%252Fimage.png%3Falt%3Dmedia%26token%3Df5fa40d4-2ad0-4777-ac47-8a88d0118044\&width=300\&dpr=4\&quality=100\&sign=738b78c8\&sv=1) ![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FkTUMfwljocZI7Rz9bbuC%252Fimage.png%3Falt%3Dmedia%26token%3D24ae2897-9893-49c2-b36b-a5a9e948d4c7\&width=300\&dpr=4\&quality=100\&sign=b69f98a3\&sv=1)

An example of a complete staking request would be as the following image:

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252F3q9mp8V6muRQjON5GnOh%252Fimage.png%3Falt%3Dmedia%26token%3Da528525b-d37e-4f22-a997-acfa205b619b\&width=300\&dpr=4\&quality=100\&sign=6e9992dc\&sv=1)

Click "Stake" to continue.

**Step 3**: Check the information and confirm your staking request by clicking "Approve".

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252F3xVxjPGXJatcdtJx8om1%252Fimage.png%3Falt%3Dmedia%26token%3D44cb0759-9462-40e2-b1a9-5abb54c4d370\&width=300\&dpr=4\&quality=100\&sign=8d1ba520\&sv=1)

**Step 4:** The result is in!

You can see your staking transaction in the History tab.

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252Fc87XJ1L1sNFMTBX9WHXP%252Fimage.png%3Falt%3Dmedia%26token%3D5b6a1b7a-7002-4724-8801-10d3813df591\&width=300\&dpr=4\&quality=100\&sign=fc8eb843\&sv=1)

If you click "View transaction", the wallet will show you the latest transaction record in your transaction history which corresponds with the extrinsic hash of this action.

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FFMJQ8PZYUFGWhMZ0nWRB%252Fimage.png%3Falt%3Dmedia%26token%3Dfa0595ef-7fc3-4e26-b492-7ce6e9e5a1e0\&width=300\&dpr=4\&quality=100\&sign=eae96e80\&sv=1)

After completing the transaction, you can see the nomination details by clicking on the respective token in the Staking tab:

![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252FfDG8fw0pyxEBqugxOxJw%252Fimage.png%3Falt%3Dmedia%26token%3Dd7409b9b-76d1-4804-8936-a6762e01b904\&width=300\&dpr=4\&quality=100\&sign=ebb959db\&sv=1)![](https://docs.subwallet.app/\~gitbook/image?url=https%3A%2F%2F631687399-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Lh39Kwxa1xxZM9WX\_Bs%252Fuploads%252Fzmnqi0yXnd5ZEg4ZyzYO%252Fimage.png%3Falt%3Dmedia%26token%3D66d33cfc-b1f8-41ad-be61-50e112da17d1\&width=300\&dpr=4\&quality=100\&sign=1e731f5b\&sv=1)

To get to the transaction history tab, please follow this [guide](https://docs.subwallet.app/main/extension-user-guide/view-transaction-history).
